如何在Github上搭建Octopress博客

Jun 24th, 2014

看了网上各种教程，都说很简单，我捣鼓了3天才整出来。 期间遇到各种网上都找不到答案的问题，在此记录下过程，以便参考。 由于刚接触，Markdown语法也不是很清楚，估计写完排版会很丑。。 看到此教程，应该是对Github和Octopress有个大概的了解了，详 细的介绍自己百度（最近Google又上不去了？不喜欢翻墙）。 下面开始，一步一步来应该是没有问题的。

<!—more—>

一、本地环境搭建（Win7 32位系统）

1.安装Git

windows 环境下安装 Git，百度 msysgit 自行下载，一步步Next即可。。

2.安装Ruby

一般推荐的版本Ruby1.9.3以上，安装时勾选 Add Ruby executables to your PATH。
将Ruby加入系统环境变量，完成后打开cmd执行：ruby-v 输出版本号表示成功。

3.安装DevKit

下载完成，解压即可。这里以解压到F:\DevKit为例，
打开cmd执行 
F:
cd DevKit
ruby dk.rb init
ruby dk.rb install

4.配置Windows系统环境变量

为了支持UTF-8编码，开始-计算机-属性-高级系统设置，添加两个变量，LANG和LC_ALL，值都是 zh_CN.UTF-8

5.更换Ruby更新源

有人说更新比较慢，我的没更换感觉也挺快的，打开cmd，执行：
gem sources -a http://ruby.taobao.org/
gem sources -r http://rubygems.org/

环境基本上就搭好了，下面开始搭建博客。。

二、搭建博客系统

1.克隆Octopress库

找一个存放Octopress的目录，打开Git Bash（右键鼠标应该可以看到），执行：
git clone git://github.com/imathis/octopress.git

完成之后，执行：
cd octopress
gem install bundler
bundle install

2.创建Github Repository

没有Github得先注册帐号，验证邮箱，之后New Repository ，注意：
Repository name 必须是 your_username.github.io (这里your_username替换成你自己的
帐号，以前的可能是 your_username.github.com ， 现在官方推荐 your_username.github.io ）
下面选项默认即可，点击 Create Repository 之后 ，什么都不要做了。

3.设置 SSH keys

绑定Github与本地电脑，不用每次都输入用户名密码，如果不设置，后面可能出现错误。
首先，本地创建一个SSH key ， 打开cmd执行：
ssh-keygen -t rsa -C"your_email@your_email.com" （就是你的邮箱啊）
按照提示输入即可。完成之后，到c:\Users\用户名.ssh\目录找到id_rsa.pub，并用文本编辑打开复制全部。

然后，到github网站右上角点击“Account settings”>>“SSH Keys”>>“Add SSH key”，将刚才复制的内容粘贴到key文本框内，
Title任意。

最后，测试一下。打开cmd执行：
ssh -T git@github.com

4.设置个人信息

$ git config --global user.name "your_username"
$ git config --global user.email "your_email@your_email.com"
请自行替换 your_username 为 你注册的帐号，替换 your_email@your_email.com 
为 你验证的邮箱。

三、发布博客到Github
